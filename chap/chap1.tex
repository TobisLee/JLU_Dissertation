\section{撰写时的一些注意事项}
\begin{itemize}
	\item 少用我、我们、we，一定要避免大白话
	\item 句与句、段与段、章与章之间的衔接注意逻辑，要避免堆砌
	\item 参考文献体例不一致，任何有缺页码、期号、卷号和字母大小写不一致的情况都会被问到怀疑人生。
	\item 你以为你的图片过关了嘛，做好画第二次的觉悟。
	\item 参考文献要新，要高引，要权威
	\item 预印本，网站，网页尽量不要引用
\end{itemize}
